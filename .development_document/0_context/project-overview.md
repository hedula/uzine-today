# 專案概覽

## 名稱

**隅誌今時（uZine Today）**

## 定位

輕量、可擴充的內容出版平台，供個人或小型編輯團隊使用的繁體中文部落格系統。

## 核心目標

- Astro 高效能內容前台
- Editor.js block-based 編輯
- Cloudflare D1 / R2 / KV 作為資料基礎設施
- 簡潔的草稿 → 發布流程

## 使用者

- **讀者**：瀏覽文章、分類、標籤
- **管理員**：單一使用者（專案擁有者），負責撰寫與發布
